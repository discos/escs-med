ESCS 0.3 CONFIGURATION AND SETUP
================================
Welcome to the documentation of escs setup for Medicina radiotelescope.
Escs is the *Enhanced single dish control system* which permits the
radiotelescope to be used by astronomers in single dish observations. The system
is the analogous of Nuraghe, installed at SRT and will be installed at Noto site
as soon as possible enabling the same kind of operations in all italian
radiotelescopes.
